Informasi SQL Injection

Apa itu SQL Injection?

SQL Injection adalah teknik penyerangan dengan menyisipkan perintah SQL ke input aplikasi, memungkinkan penyerang mengakses database.

Jenis-jenis SQL Injection:

  • Classic SQLI
  • Blind SQLI
  • Time-based Blind SQLI
  • Union-based SQLI

Cara Mencegah:

  • Gunakan prepared statements
  • Validasi input pengguna
  • Prinsip least privilege untuk database user
  • Enkripsi data sensitif
